## My personal Mac env setting

#### useage

clone this repo and then cd into it.
Edit your ~/.zshrc (I'm using zsh ,but your don't have to) and adding the following lines:

```bash
export dotDir=$HOME/dots
export dotZshrc=$dotDir/zshrc
. $dotZshrc/zshrc.sh
```


```bash
$ cd ~/dots
$ ./init.sh
```

all set !
